our next data structure is called an object. objects, unlike arrays have no built in order, and instead store data in key-value pairs.

    var person ={
  • name: "travis",
  • age:21'
  • city: "LA"
  • };

in order to retrieve data from an object you can use dot notation

    IE
  • person.name
  • person.age
  • person.city

in order to update an object you simply need to access the data and then reassign it

    person.age +=1
  • person.age now === 23
