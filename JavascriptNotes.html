Javascript Notes!

5 primitive data types in javascript are Numbers, Strings, Booleans, null, and undefined

    These data types are easy to differentiate:
  • Numbers:
    • 4
    • 9.3
    • -10
  • Strings:
    • "Hello World"
    • "43"
  • Booleans:
    • true
    • false
  • null and undefined
    • null
    • undefined

escape characters are useful in strings to tell javascript to include characters that are usually functional

    ie:
  • "singin \"do wah diddy\"" would return as
  • singin "do wah diddy"

strings have a length property (.length) which can be accessed like an index: []

  • "hello".length
  • returns: 5
  • and "hello"[3]
  • returns: l
  • "hello"[0]
  • returns: h

variables follow a specific format of "var yourvariablename = yourvalue;"

  • var name = "rusty";
  • var secretNumber = 8611;
  • var isAdorable = true ;

null and undefined are really values variables can be set to.

  • null is used to explicitly declare something as having no value or being empty
  • undefined is used to mark something as empty "so far"

3 methods useful in javascript are alerts, prompts and console.log

  • alert:
    • alert("Hello There!!!")
    • = Hello There!!!
    • alert(3*4)
    • = 12
  • console.log:
    • console.log("hello from the console!")
    • = hello from the console!
    • but is only printed in the console
  • prompt:
    • prompt("What is your name?")
    • asks the user for the information
    • which can be stored in a var like this
    • var userName = prompt("What is your name?")

conditional statements in javascript!!!

    Comparison operators are the lifeblood of conditionals

  • greater than: >
  • greater than or equal to: >=
  • less than: <
  • less than or equal to: <=
  • equal to: ==
  • not equal to: !=
  • equal value and type: ===
  • not equal value or type: !==

    Logical Operators

  • AND: &&
    • 5 < 10 && 5!==5
    • returns false
  • OR: ||
    • 9 > 9 || 5 === 5
    • returns: true
  • NOT: !
    • !(5===9)
    • returns: true

    The only inherently false Values in javascript are

  • false
  • 0
  • ""
  • null
  • undefined
  • NaN
    • Everything else is "Truthy"!!!

loops are important in writing DRY code

    types of loops:

  • while loops run while a condition is true
      this will define a var and print it into the console while counting until it reaches a given value.
    • var count = 1
    • while(count < 6) {
    • console.log("count is: " + count);
    • count++;
    • }
  • for loops run under specific sets of circumstances:
    • for(initiator; condition; step){ }
        printing number 1-5 with a for loop:
      • for(var count=0; count<6; count++) {
      • console.log(count);
      • }
      it is important to note that variables created inside a for loop exist only inside the loop!

indexOf is important for checking to see if words are present at all in string.

    while (string.indexOf("yes")===-1 && string.indexOf("yeah")===-1)
  • remember index of a string starts at 0
  • and javascript reads values not present in a string as -1
  • says while yes does not exist in the string AND yeah does not exist in this string
  • so this while loop will continue until the string contains either yes, or yeah specifically.
