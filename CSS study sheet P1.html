CSS stands for cascading style sheets. often considered the adjectives to html's nouns.

CSS has a general rule to follow.

  • selector {
  • property: value;
  • another property: value;
  • }

    Types of css selectors:
  • Element:
    • li{
    • ...
    • }
  • Class:
    • .hello{
    • ...
    • }
  • id:
    • #name{
    • ...
    • }
  • Star:
    • *{
    • ...
    • }
  • Descendant Selector:
    • li a {
    • ...
    • }
  • Adjacent Selector:
    • h4 + ul {
    • ...
    • }
  • Attribute Selector:
    • a[href,img,checkbox, etc.]{
    • ...
    • }
  • nth of type:
    • ul:nth-of-type(x or even or odd){
    • ...
    • }

Inheritence and Specificity

lacking a specific css styling, a styling of a parent element will be inherited by its children elements

similarly, the more specific the stylings target the more precedence it is given.

ie if you give the body a background color and then give a div a different background color the div will gain the new color

the more specific styling of the div has taken over.
