Dom Manipulation

What is the DOM? DOM stands for Document Object Model. this is the interface between the javascript and your HTML/css.

The web broser opens the entirety of your code in a Document. this contains the body and every bit of css and javascript modifying it, and it is represented as an object with all the attributes of each piece of your webpage.

The process for manipulating the DOM is select and manipulate.

    Imagine you want to change the color of your header.
  • var h1 = document.queryselector("h1");
    • select the header and save it to a variable
  • h1.style.color = "pink"
    • Manipulate the text to change it's color to pink.
