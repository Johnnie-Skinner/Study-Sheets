up until now we have used console.log to print our returned values into the console. This is only useful to test outputs. Much more useful is the return funtion.

    return:
  • function square(x) {

  • }

Higher order functions are functions that either have other functions passed to them or return other functions.

    defines a function that will count up as long as it hasnt reached 100 but only once when called:
  • var count = undefined
  • function countToHundred() {
  • if (count === undefined) {
  • count = 0;
  • console.log(count+1);
  • count++;
  • }
  • else if (count <= 99) {
  • console.log(count+=1);
  • }
  • else {
  • console.log("finished!")
  • }
  • }
    calls the function above once every 1000ms or once a second:
  • setInterval(countToHundred,1000)

an array is a data structure javascript uses to store information.

instead of writing 4 seperate variables you can write an array: var numbers = [1,2,3,4];

this array is automatically indexed starting with zero and to access or change the array you use index notation:

  • to access the above array:
  • friends[1]
  • returns: 2
  • and to change or add data:
  • numbers[4] = 5
  • which in this case adds 5 to the array.

built in methods for arrays:

  • push/pop
    • push adds to the end of an array:
        to add green to the end of an array:
      • colors.push=("green")
    • pop removes the last element of the array
      • colors.pop()
      • will return the last item in the array and then remove it from the array
  • shift/unshift
    • adds and removes from the beginning of the array but function the same as push/pop
    • unshift ADDS and shift removes (think british)
  • indexOf
    • friends.indexOf("David")
    • will find the first instance of David in the array "friends" and return its index
  • slice
    • is used to copy portions of an array
    • var citrus = fruits.slice(1,3);
    • slices out index 1 and 2 of the array fruits and puts it in a new array "citrus"

the .forEach's generic internal function requires one but can take up to 3 arguments. these are in a specific order:

    array.forEach(function(E,I,arr){...});

  2. I - represents the index of said element
  3. arr - represents the array that .forEach was called on
