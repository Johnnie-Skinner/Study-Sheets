Arguments are how we can write functions that take inputs.

    in this example the function "square" receives input "x" and prints the result of its square in the console
  • function square(x) {
  • console.log (x*x);
  • }
    • where x is the argument of the function and is usually defined by either input or another funciton.

functions can accept more than one argument as well

  • function area(length,width) {
  • console.log(length*width);
  • }

up until now we have used console.log to print our returned values into the console. This is only useful to test outputs. Much more useful is the return funtion.

    return:
  • function square(x) {
  • return (x*x)
  • }
    • this will not only perform the function but actually pass that value out as a "return"

Higher order functions are functions that either have other functions passed to them or return other functions.

    defines a function that will count up as long as it hasnt reached 100 but only once when called:
  • var count = undefined
  • function countToHundred() {
  • if (count === undefined) {
  • count = 0;
  • console.log(count+1);
  • count++;
  • }
  • else if (count <= 99) {
  • console.log(count+=1);
  • }
  • else {
  • console.log("finished!")
  • }
  • }
    calls the function above once every 1000ms or once a second:
  • setInterval(countToHundred,1000)
