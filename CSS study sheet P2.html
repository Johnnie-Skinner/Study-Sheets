Manipulating font in css!

  • font-family:
    • p{
    • font-family: "Arial"
    • }
  • font-size:
    • p {
    • font-size: 20px or xem
    • }
    • "em" is relative size to the parent element
    • 2.0em would be double the size of the parent element
  • font-weight:
    • p{
    • font-weight: bold or normal or numeric 100-800 by 100's
    • }
  • Line Height:
    • p{
    • line-height: 2;
    • }
    • multiplies the current line height by the given value
  • text-align:
    • h1{
    • text-align: center;
    • }
  • text-decoration:
    • p {
    • text-decoration: underline;
    • }

Now we find that we want to use custom fonts that maybe machines dont have by default. for this we will use google-fonts to start. other resources are valid but in any case we need to embed a link tag to the specific font. it can be local or web pased and should be included in the head. i.e. link href="https://fonts.googleapis.com/css?family=Charm:400,700" rel="stylesheet"

Box Model- In a document, each element is represented as a rectangular box. In CSS, each of these rectangular boxes is described using the standard box model. Each box has 4 edges: the margin edge, the border edge, the padding edge, and the content edge.

    Adjusting basic block properties:
  • width: x or %(of parent element)
  • height: x or %
  • border: _px type color
  • padding: x or %
  • margin:
  • border, padding and margin can be modified with -left, -right, etc.
  • margin: top right bottom left;
  • modified all 4 margins differently on one line

margin: auto; is commonly used to center an element content in the existing space. it equalizes the margins

another way to modify the margin looks like margin: 20px auto; modifying the top and bottom together and the left and right together respectively

float: can be used to move images or elements out of the normal flow to form a grib like structure
